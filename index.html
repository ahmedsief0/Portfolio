My Projects

Digital Prayer time clock

The project is a smart prayer time and clock system built with an ATmega32 microcontroller, DS3231 RTC, seven-segment displays, and an ISD1820 module for Adhan playback. It features automatic prayer time calculation using astronomical equations, Bluetooth configuration via a mobile app, and a touch sensor for dismissing the alarm. A power-saving mode with a PIR motion sensor ensures efficient energy use by activating the display only when movement is detected.

Attendance System Using (face recognition)

The project is a real-time face detection and recognition attendance system that identifies individuals from a live camera feed using the face_recognition library (dlib with HOG/CNN). It features automatic attendance logging with timestamps into CSV/Excel databases, prevents duplicates, and supports report generation (daily, weekly, monthly). An optional GUI dashboard allows user registration, log viewing, and admin control, with high recognition accuracy and potential anti-spoofing measures.

Mobile Temperature controlled fan

The project is an automatic fan control system that adjusts fan speed based on ambient temperature using a sensor and PWM, ensuring efficient cooling. It also supports manual control via a Bluetooth mobile app, allowing users to turn the fan ON/OFF, set speed, and monitor temperature in real time. The system provides dual modes (Auto/Manual) with display output for temperature, fan speed, and control status.

Virtual router solution for multinational school on Cisco packet tracer

The project is a multi-branch school network design connecting UK HQ with Asia, Africa, and Cairo via WAN Point-to-Point links, ensuring full branch-to-branch communication. It includes an IP addressing scheme, static/dynamic routing, and end devices (PCs, servers, printers) with proper device naming for easier management. Advanced features such as VLAN segmentation (Admin, Teachers, Students) and Router-on-a-Stick provide scalability, security, and organized traffic flow.
